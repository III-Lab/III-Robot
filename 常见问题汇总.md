

1. 使用串口重定向后，脱机无法工作

   ```c
   int fputc(int ch, FILE *f)
   {
   	HAL_UART_Transmit(&huart3, (uint8_t *)&ch, 1, HAL_MAX_DELAY);
   	return ch;
   }
   
   int fgetc(FILE *f)
   {
   	uint8_t ch = 0;
   	HAL_UART_Receive(&huart3, &ch, 1, HAL_MAX_DELAY);
   	return ch;
   }
   ```

   **解决：✔Use MicroLIB**
